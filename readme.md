# RDA Vedanta - Corporate Website

## Project Overview
High-conversion B2B homepage for an electrical manufacturing enterprise based in Lucknow.

## Tech Stack
- HTML5 (Semantic)
- CSS3 (Custom Properties / Flexbox / Grid)
- Vanilla JavaScript

## Folder Structure
- `src/css`: Stylesheets using BEM naming convention.
- `src/assets`: Product images and downloadable catalogs.
